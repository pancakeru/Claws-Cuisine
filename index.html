Design Statement

I decided to make a simple procedurally generated side scroller, with the theme being a spin off of another game I had developed in Unity. I chose this because oftentimes the best way to learn a new engine and language is to try and recreate a past project using that new engine.

The core gameplay loop of this game involves first collecting meat in the platformer, then cooking and selling it for money, before purchasing upgrades that help the player get stronger in the platformer so they can collect more meat. I felt that this positive feedback loop offered a fun and intuitive flow to the game.

I wanted to capture a lighthearted and cozy feel, hence why I ensured the sprites were brightly colored and the background music was cheerful. The biggest challenge for me was to think of an interactive way to utilize different HTML elements on the page, as well as playtesting the features that involved local storage. Overall, I think this game turned out quite well. While it is not as complex as I would have liked it to be, I think it performs well as a web game.
